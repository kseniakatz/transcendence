# ft_transcendence

Real-time multiplayer Pong web application built in a team of 3.

---

## 🚀 Overview

This project is a full-stack single-page application where users can play Pong online in real time.

The focus was on building a responsive frontend and handling real-time communication between players.

---

## 🛠 Tech Stack

**Frontend**
- React 19 + TypeScript
- Tailwind CSS
- Zustand (state management)

**Backend**
- Node.js
- REST API + WebSockets

**Infrastructure**
- Docker (microservices architecture)

---

## ⚡ Features

- Real-time multiplayer gameplay (WebSockets)
- Authentication with JWT (HttpOnly cookies)
- Game flow: lobby → game → result
- Responsive UI with reusable components

---

## 👩‍💻 My Role

- Built the frontend SPA using React and TypeScript  
- Integrated WebSocket communication for real-time gameplay  
- Connected frontend to backend APIs  
- Designed reusable UI components (custom design system)
