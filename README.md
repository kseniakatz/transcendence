# ft_transcendence

Full-stack real-time Pong web application built in a team of 3

## Tech stack
- React 19 + TypeScript + Tailwind CSS
- Node.js backend
- WebSockets (real-time gameplay)
- Docker (microservices architecture)

## Features
- Real-time multiplayer
- Authentication (JWT cookies)
- Game flow: lobby → game → result

## My role
- Frontend (SPA, UI system)
- WebSocket integration
- API integration
